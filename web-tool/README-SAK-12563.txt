tool registrations:

  sakai.iframe 
  sakai.iframe.myworkspace 
  sakai.iframe.service 
  sakai.iframe.site

tool identifier.   This tool id could also be moved to the web-portlet
but the code for annotatedurl needs to be completed and tested.
