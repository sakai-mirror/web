Adicionar, editar, ou deletar um item do Conteúdo Web

Você pode adicionar um link para um site externo, ou um item de Conteúdo da Web, no menubar do seu site de curso ou projeto, se você tiver as permissões apropriadas. Qualquer um pode adicionar um item de Conteúdo da Web em Meu Workspace.

Para obter instruções adicionais sobre a edição ou remoção de itens de seu site, veja Adicionar, editar, ou remover ferramentas.

Nessa página:

  • Adicionar um item de Conteúdo Web em Meu Workspace
  • Editar um item de Conteúdo Web
  • Deletar um item de Conteúdo Web

Adicionar um item de Conteúdo Web no menubar do seu site de curso ou projeto

Nota: Você precisa ter as permissões apropriadas para adicionar um item de Conteúdo Web no site de curso ou de projeto.

  1. No site do curso ou projeto relevante, clique em Site Info.

  2. Na página Site Info, clique em Editar Ferramentas.

  3. Marque a caixa próxima do Conteúdo Web. (Se o seu site já contém itens de Conteúdo da Web, eles serão listados com o título que você deu a eles, ou "Conteúdo da Web" se você não mudar o título, mas você deve ver uma caixa adicional vazia rotulada Conteúdo Web; marque ela.) Clique em Continuar.

  4. Na tela de "Adicionar Múltiplas Instâncias de Ferramentas", digite as seguintes informações:

    • No campo "Título:", adicione um título.
    • No campo "fonte", digite uma URL depois do "http://" que já aparece no campo.
    • Se você gostaria de adicionar mais de um item, selecione o número apropriado de itens no menu Mais Ferramentas Conteúdo Web?.

  5. Clique em Continuar. Você verá informações sobre as alterações feitas em seu site. Clique em Finalizar, ou, se você deseja fazer mais alterações, clique em Voltar. Clique em Cancelar para cancelar suas alterações.

Adicionar um item de Conteúdo Web em Meu Workspace

  1. Em Meu Workspace, clique em Configurações do Worksite.

  2. Na página de Configurações do Worksite, você verá todos os seus sites listados. Para adicionar um item ao Conteúdo Web no Meu Workspace, marque a caixa próxima ao Meu Workspace. Para adicionar um item no Conteúdo Web em um dos sites que você gerencia, marque a caixa próxima ao nome do site.

  3. Clique em Editar. Na próxima página, clique em Editar Ferramentas.

  4. Marque a caixa próxima do Conteúdo Web. (Se o seu site já contém itens de Conteúdo da Web, eles serão listados com o título que você deu a eles, ou "Conteúdo da Web" se você não mudar o título, mas você deve ver uma caixa adicional vazia rotulada Conteúdo Web; marque ela.) Clique em Continuar.

  5. Na tela de "Adicionar Múltiplas Instâncias de Ferramentas", digite as seguintes informações:

    • No campo "Título:", adicione um título.
    • No campo "fonte", digite uma URL depois do "http://" que já aparece no campo.
    • Se você gostaria de adicionar mais de um item, selecione o número apropriado de itens no menu Mais Ferramentas Conteúdo Web?.

  6. Clique em Continuar. Você verá informações sobre as alterações feitas em seu site. Clique em Finalizar, ou, se você deseja fazer mais alterações, clique em Voltar. Clique em Cancelar para cancelar suas alterações.

Você deverá ver agora um link no menu para o item de Conteúdo Web com o título que você forneceu.

Editar um item de Conteúdo Web

Para editar um item de Conteúdo Web existente:

  1. No menubar do seu site, clique no título do item de Conteúdo Web apropriado.

  2. Perto do topo do frame Conteúdo da Web, clique em Opções.

  3. No formulário "Personalizar Conteúdo Web", você pode alterar:

    • Título da Ferramenta: O título que aparece na parte superior do frame Conteúdo da Web
    • Título da Página: O título que aparece na barra de menu do seu site
    • Altura do Frame: A altura (em pixels) do frame de Conteúdo Web
    • URL: a URL para o item do Conteúdo Web

  4. Para fazer com que o item de Conteúdo Web abra em uma nova janela do navegador, marque a caixa ao lado de Abrir em uma nova janela? .

  5. Quando terminar, clique em Atualizar Opções para salvar suas alterações, ou Cancelar para cancelar suas alterações.

Deletar um item de Conteúdo Web

  1. Para deletar um item de Conteúdo Web, clique em Meu Workspace, e clique em Configurações do Worksite.

  2. Se você deseja remover um item de Conteúdo Web de um site de curso ou projeto, marque a caixa próximo ao nome do site. Se você deseja remover um item de Conteúdo Web do Meu Workspace, marque a caixa próxima do Meu Workspace.

  3. Clique em Editar. Clique em Editar Ferramentas.

  4. A partir da lista de ferramentas, desmarque o item que deseja excluir e, em seguida, clique em Continuar. Reveja sua informação, e clique em Finalizar para deletar o item.

Nota: Se o item de Conteúdo Web ainda aparecer no menu do site, clique emInício para atualizar o menubar.
