Conteúdo Web: Resumo

O que faz

A ferramenta Conteúdo Web permite que os proprietários do site escolham web sites externos para mostrar dentro de seus worksites.

Principais Conceitos

Links: Para criar um link do Conteúdo da Web, você pode especificar a URL de qualquer site da web e um breve título descritivo. O link aparecerá no menubar com o título que você escolheu; quando você clicar no link, o website abrirá dentro do seu worksite. Você pode adicionar múltiplos links Conteúdo da Web para sua barra de menu.

Considerações

  • Se você deseja fornecer muitos links paraos participantes de seu curso ou do projeto, você pode evitar congestionar o seu menu listando a maioria deles na seção Repositório.

  • Departamentos, colaborações ou outros grupos podem usar a ferramenta Conteúdo da Web para fornecer um link de destaque para seus worksites públicos.
