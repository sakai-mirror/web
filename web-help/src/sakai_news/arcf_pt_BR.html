Notícias: Resumo

O que faz

A ferramenta de Notícias usa RSS para trazer notícias dinâmicas ao seu worksite. RSS é um formato de dados usado para notícias na web. Ele permite que o conteúdo seja continuamente atualizado a partir de um site de notícias remoto para aparecer em outro site.

Principais conceitos

RSS: Para configurar o seu worksite para exibir notícias, você deve especificar uma URL apontando para um feed de notícias RSS. Um feed de notícias RSS é uma lista especialmente formatada de artigos de notícias. Você pode encontrar feeds RSS usando um mecanismo de busca na Internet (por exemplo, o Google ou o Yahoo). Sites que distribuem notícias no formato RSS são chamados de "feeds de notícias". A ferramenta de Noticias RSS é um "agregador de notícias" (ou "leitor de notícias") que lê e exibe o conteúdo de RSS feeds de notícias.

Considerações

  • Você pode adicionar mais de um feed ou quando você configurar o site ou num momento posterior.

  • Além de notícias, alguns sites usam o formato RSS para informação é atualizado com frequência, tais como blogs (web logs que servem como diários pessoais acessíveis ao público), listagens de eventos ou o histórico de revisão de um livro.
